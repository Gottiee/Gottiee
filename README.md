**Glasck** est une plateforme web ambitieuse pour le suivi de l’esport League of Legends, proposant une interface claire, esthétique et intuitive, avec un suivi en temps réel des matchs, équipes, ligues et joueurs, ainsi que de nombreuses statistiques.
